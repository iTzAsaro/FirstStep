# Stack Tecnológico

Este documento resume las tecnologías principales utilizadas en el proyecto y su propósito dentro de la solución.

## Frontend

- Framework UI: React
- Bundler/dev server: Vite
- Lenguaje: TypeScript
- Estilos: Tailwind CSS

## Backend

- Runtime: Node.js
- Framework: Express.js
- Lenguaje: TypeScript

## Base de Datos

- Motor: Oracle Database
- Driver: oracledb

## Seguridad

- Autenticación/autorización: JWT
- Hash de contraseñas: bcryptjs
- Mitigación de ataques comunes: sanitización frente a XSS y SQL Injection

## DevOps y Herramientas

- Contenedorización: Docker
- Control de versiones / repositorio: GitHub
- Pruebas y documentación de APIs: Postman
